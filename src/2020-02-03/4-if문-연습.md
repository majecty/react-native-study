# if문-연습

다음 조건을 직접 만들어보자.

똑또한 할일 관리 앱이 중요한 일정을 알아서 추천해주는 기능을 넣었다.
조건에 따라 각 할일의 중요도를 결정해보자.

## 스펙

남은 시간이 30분 이내거나, "교수님" 태그가 붙어 있는 일은 "매우 중요한" 일이다. 남은 시간이 3시간 이내인 일은 "중요한" 일이다. 나머지 일은 "덜 중요한" 일이다.

```js
let leftTimeMinutes = 20;
let tag = "교수님";

if (true) {
  console.log("매우 중요");
} else if (true) {
  console.log("중요");
} else {
  console.log("덜 중요");
}
```
